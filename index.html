Introduction

The Saline Machine, developed in 2023 under the guidance of Mr. Frederick, a lecturer at Temasek Polytechnic's Advanced Manufacturing Centre, is a remarkable achievement. This project successfully replicates an industrial saline production system with precision. Equipped with advanced data analysis tools, it enables continuous performance monitoring and environmental impact assessment, ensuring maximum operational efficiency.

Purpose of this Project

The saline production machine is a multi-disciplinary project that integrates electronics, mechatronics, computer engineering, and biomedical engineering. Its purpose is to provide students with a comprehensive understanding of how a manufacturing system operates on a production line. The system is designed to allow roles such as operations managers and sustainability officers to track key performance indicators, including Overall Equipment Effectiveness (OEE) and sustainability metrics. While the machine closely replicates a real-world production system, further improvements are necessary to fully align it with industrial standards and ensure it functions as an actual industry-grade system.

Key Components of the Saline Machine

Water Purification System

Removes impurities and contaminants from the water source to ensure sterility.

Salt Dissolving Unit

Precisely dissolves sodium chloride to create the desired concentration of saline solution.

Filtration System

Removes any remaining particulate matter to ensure the solution's purity.

Packaging System

Fills, seals, and labels the saline solution containers for safe and sterile storage.

Operation Guide

Shift Instructions for Standardization:

For Maintenance Check:

  1. Turn on the machine via the power supply. The OEE “Stop Duration” will start running as the machine automatically goes to “Stop” mode.
  2. Switch to “Manual” mode. Empty all the water in the pipes by opening all valves and using air pressure. The “Manual Duration” will start running.
  3. Empty all the water in the mixing bin, stock bin, and diluent bin by opening all relevant valves. Close the valves once emptied.
  4. Set the machine to “Stop” mode. Empty the waste bin and refill the water supply tank with clean water.
  5. Switch back to “Manual” mode and test the functionality of all valves and mixers.
  6. Return to the “Main Page,” navigate to “Ideal Page,” and input the relevant values.

For Stock Solution Testing:

  1. Go to the “Main Page” and select “Stock Tank.”
  2. On the “Quality Check (Stock Tank)” page, follow Step 1 by clicking on “Testing.”
  3. Proceed to Step 2 and click “Auto” to fill both the diluent and stock tanks.
  4. When the “Ready for Concentrate” indicator lights up in Step 3, add the concentrate.
  5. Click the “Concentrate Successfully Added” button. A portion of the stock solution will flow to the waste bin, where the conductivity sensor will test the solution.
